# CRUD Products Python

Proyecto CRUD de productos usando Python y archivos JSON.

## Funciones

- Crear producto
- Leer productos
- Actualizar producto
- Eliminar producto

## Pipeline CI/CD

Se ejecuta automáticamente cuando se hace push al repositorio.

Incluye:

- Instalación de dependencias
- Pruebas con pytest
- Verificación de código con flake8
